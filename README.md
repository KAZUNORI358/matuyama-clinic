# マツヤマクリニック (架空サイト)

## 概要

Hello Mentor の課題でFigmaのデザインカンプをもとに、コーディングしたLPです。

## 制作期間

2025年12月(約1週間)

## 公開URL

[matuyama-clinic.kazunori-folio.com/](https://matuyama-clinic.kazunori-folio.com/)

## 概要

コーポレートサイトです。  
TOPページのみのコーディングです。

## Cursor 推奨拡張機能

開発効率を向上させるために、以下の拡張機能の使用しています

### SCSS 開発

- **Live Sass Compiler**： SCSS ファイルのリアルタイムコンパイル

### コードフォーマット

- **Prettier**： HTML、SCSS、JavaScript、JSON 等のコードフォーマット

## ファイル構成

```
Masuyama-Clinic/
├── .vs-code/
├── css/
├── img/
├── js/
├── scss/
├── index.html
└── README.md
```

## コーディングルール

- **命名規則**: FLOCSS (c-,l- プレフィックスを使用)
- **CSS**: Sass(SCSS) を使用
- **REM単位**: 文字サイズ、余白の指定にrem (root em) 単位を使用しています。
